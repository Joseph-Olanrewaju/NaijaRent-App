# NaijaRent-App
NaijaRent is a full-stack Nigerian rental marketplace built to eliminate the most common frustrations of house hunting in Nigeria: fake listings, hidden agent fees, cash-only transactions, and the lack of any formal tenant protection. The platform connects verified landlords, professional agents, affiliate agents, and responsible tenants within one secure, transparent digital environment.

The application is delivered as a single optimised HTML file that runs entirely in the browser — no server installation or technical setup is required. It comes pre-loaded with 15 verified demo listings across 10 Nigerian cities and 8 demo user accounts representing every role on the platform.

2.  User Roles
NaijaRent supports four distinct user types. Each has a dedicated dashboard, a tailored set of features, and a separate registration path.

2.1  Tenants
Tenants are the primary users of NaijaRent. During registration, they provide their username, NIN (National Identification Number), next-of-kin details (name, relationship, phone, and address), employment and salary information, and their preferred area and property type.

Once logged in, tenants can:
Browse and filter verified property listings by city, type, price, and occupancy status
View AI-powered house recommendations personalised to their income and preferences
See both annual and monthly rent breakdowns on every listing
Chat with RentRec, the built-in AI financial advisor, before making any decision
Apply for properties and message landlords directly through the platform
Track application status and make secure rent payments once approved
Save favourite listings, view payment history, and file complaints when needed

2.2  Landlords
Landlords register with their bank account details so that rent payments can be processed and disbursed directly. They can:
Create property listings with photos, videos, description, price, and amenity details
Review incoming tenant applications — including NIN (last 4 digits) and next-of-kin information
Approve or reject applicants and communicate with them through the messaging system
Toggle a property's occupancy status between Occupied and Vacant
Track total earnings across all their listings
Receive 90% of every rent payment; NaijaRent retains a 10% platform fee

2.3  Agents
Agents have the same capabilities as landlords but manage listings on behalf of multiple property owners. They receive a verified agent badge after admin approval and have access to a full earnings tracker.

2.4  NaijaRent Affiliate Agents
The affiliate agent programme allows any Nigerian adult (18 and above) to earn income by identifying and listing vacant houses in their community. Registering as an affiliate agent requires a NIN, date of birth, gender, state of origin, LGA, workplace, bank account details, a profile photo, and acceptance of the platform terms.

Once verified by admin, an affiliate agent can list properties. A 15% fee is automatically added to all their listings: 10% is paid to the agent on every successful rent transaction, and 5% goes to the NaijaRent platform.
